# portfolio

  • ## ms-equipamento (docs) - Microsserviço que gerencia equipamentos de um bicicletário

    Faz parte de um serviço backend de um Bicicletário projetado em 3 microsserviços isolados: Aluguel, Equipamento e Externo. O Microsserviço Equipamento foi desenvolvido em Node.js, sendo responsável por manter e gerenciar bicicletas, trancas e estações.

    É integrado com o Microsserviço Aluguel (microsservicoAluguel), que é responsável por gerenciar os aluguéis de bicicletas e os usuários, e com o Microsserviço Externo (Externo_PM), que é responsável por se comunicar serviços externos, como o serviço de cartão de crédito e envio de emails.

  • ## PlatCon (deploy) - a svelte project

    Fullstack (svelte/postgres) uses sveltekit template I built here, which in turn was built from the svelte-mento project. (unfinished yet)

  • ## svelte-mento (deploy) - album discover app

    a svelte frontend app that just integrate a Last.fm API with a very popular tag (1001 albums to listen before you die) and daily recommends you 6 albums every day at homepage, etc.

    Note: This is a very personal project from when I started to delve into good coding practices, performance and handle with complexity that I had neglected in the rush while I needed to deliver the Ticket.io project at the right time. my first contact with typescript, unit and interface tests, redis caching, and this time using Tailwind I was breaking my head with styling more than in the previous project where I just copypasted examples from the Bootstrap documentation.

  • ## Ticket.io (deploy) - IT Problem Management Platform

    I did it in group in for Systems Design and Construction course at my university, it was one of my first contacts with web development, a fullstack Node.JS/Express application, MySQL, Firebase & React.

  • ## coelho-souls (play at itch.io) - Platform Game

    A metroidvania open-source game I made (individually) in one semester for a Game Project course at my university using the Godot 3 game engine. except music, all assets were made by me.
